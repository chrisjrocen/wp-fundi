== Description ==
WP Fundi is a modern, clean WordPress theme built following WordPress coding standards and best practices.  
It provides a responsive, accessible, and SEO-friendly foundation for building websites, with a focus on performance and customization.

== Features ==
* Responsive Design – mobile-first approach with clean, modern styling
* Accessibility – WCAG compliant with ARIA labels and keyboard navigation
* SEO Optimized – semantic HTML5 markup and meta support
* Customizable – support for custom logos, menus, and widget areas
* Performance – optimized code with minimal dependencies
* WordPress Standards – developed using WordPress coding standards and best practices

== Theme Support ==
* Post Thumbnails – featured images for posts and pages
* Custom Logo – upload and display a custom site logo
* Navigation Menus – primary and footer menu locations
* Widget Areas – sidebar and footer widget areas
* HTML5 – full HTML5 markup support
* Customizer – selective refresh for widgets
* Responsive Embeds – responsive video and embed support
* Block Editor – Gutenberg block editor styles and support

== Installation ==
1. Upload the `wp-fundi` folder to `/wp-content/themes/`
2. Activate the theme through WordPress Admin → Appearance → Themes
3. Customize the theme through Appearance → Customize

### Install via WordPress Admin
1. In the Dashboard, go to Appearance → Themes → Add New → Upload Theme
2. Select the theme ZIP file `wp-fundi.zip`
3. Click **Install Now** then **Activate**

== Customization ==
* **Colors** – edit CSS variables in `style.css`
* **Layout** – adjust grid and spacing in `style.css`
* **Functionality** – extend features in `functions.php`
* **Templates** – override parts in `/template-parts/`
